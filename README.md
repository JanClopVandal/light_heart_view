# Light Heart — Interactive Light Installation

Static case-study page. Open `index.html` or deploy via GitHub Pages.

## Deploy on GitHub Pages
1. Push these files to the root of `JanClopVandal/light_heart_view` (branch `main`).
2. Repo → **Settings** → **Pages** → Source: **Deploy from a branch** → Branch: **main / (root)** → Save.
3. After ~1 min the page is live at `https://janclopvandal.github.io/light_heart_view/`.
